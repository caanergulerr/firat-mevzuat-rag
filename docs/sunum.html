Teknoloji Yığını

Neden Bu Teknolojiler?

🐍
Python
Yapay zeka ve NLP araştırmalarının fiili standardı. BERTurk ve ChromaDB yalnızca Python ekosisteminde mevcut.
↑ Ekosistem zenginliği
⚡
FastAPI
Yüksek performanslı async REST API. Otomatik Swagger dokümantasyonu ve tip güvenliği ile hızlı geliştirme.
↑ Performans · ↑ Dokümantasyon
🇹🇷
BERTurk
Türkçe metinler için özel eğitilmiş BERT modeli. Hukuki Türkçe terimlerini genel modellere göre çok daha iyi anlar.
↑ Türkçe dil desteği
🗄️
ChromaDB
Kurulum gerektirmeyen gömülü vektör veritabanı. Cosine similarity ile milisaniyeler içinde anlamsal arama.
↑ Hız · ↑ Kolay kurulum
🔗
LangChain
RAG zincirlerini, prompt şablonlarını ve LLM entegrasyonunu birleştiren orkestrasyon çerçevesi.
↑ RAG orkestrasyon kolaylığı
⚛️
React
Bileşen tabanlı modern kullanıcı arayüzü. Gerçek zamanlı sohbet akışı ve kaynak gösterimi için ideal.
↑ Kullanıcı deneyimi
