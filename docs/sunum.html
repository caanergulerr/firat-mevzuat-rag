Neden Bu Teknolojiler?

🐍
Python
Makine öğrenmesi ve NLP araştırmalarının fiili standardı. BERTurk ve ChromaDB yalnızca Python ekosisteminde mevcut.
↑ NLP Ekosistemi
🇹🇷
BERTurk
Türkçe hukuki metinler için özel eğitilmiş BERT modeli. "Mazeret", "kayıt dondurma" gibi terimleri genel modellere kıyasla çok daha iyi anlar.
↑ Türkçe Dil Desteği
🗄️
ChromaDB
Kurulum gerektirmeyen, gömülü vektör veritabanı. Cosine similarity ile milisaniyeler içinde anlamsal arama yapar.
↑ Hız ve Kurulum Kolaylığı
⚡
FastAPI
Yüksek performanslı async REST API. Swagger UI otomatik üretilir, akademik prototip geliştirme için idealdir.
↑ Performans ve Dokümantasyon
🔗
LangChain
RAG zincirlerini, prompt şablonlarını ve LLM entegrasyonunu tek çatı altında yöneten orkestrasyon çerçevesi.
↑ RAG Orkestrasyon
⚛️
React
Bileşen tabanlı modern kullanıcı arayüzü. Gerçek zamanlı sohbet akışı ve kaynak gösterimi için idealdir.
